# 矿权日报 Agent 前端产品需求文档（PRD）

## 1. 产品概述

为「矿权日报 Agent」提供一个 React Web 控制台，定位为「Agent 运行控制台 + Markdown 报告阅读器」。用户输入自然语言任务，实时查看 LangGraph 节点执行进度，最终获取结构化矿权日报（含新闻摘要、资源量数据、价格走势、风险提示、引用源），支持复制和下载。

- **目标用户**：矿权投资分析师、矿业研究员、需要快速获取矿权项目动态的决策者
- **核心价值**：将复杂的 MCP + LangGraph Agent 编排过程可视化，让用户像使用搜索引擎一样生成专业矿权日报

## 2. 核心功能

### 2.2 功能模块

1. **首页**：Agent 输入框、示例 Prompt、最近报告列表
2. **任务运行页**：LangGraph 节点执行进度、实时运行日志、Warning 展示、任务取消
3. **报告详情页**：Markdown 报告渲染、资源量表格、价格走势图、风险提示、引用源、复制/下载
4. **历史报告页**：报告列表、搜索筛选、状态展示、查看详情/下载

## 3. 核心流程

### 3.1 用户创建任务流程

用户在首页输入自然语言指令 → 点击生成 → POST /api/briefings 创建任务 → 获取 task_id → 跳转到运行页

### 3.2 实时监听任务流程

运行页建立 SSE 连接 → 接收 node_start/node_success/node_warning 事件 → 更新节点状态和日志 → 收到 task_done → 跳转报告详情页

### 3.3 报告展示流程

报告详情页 GET /api/briefings/{task_id} → 获取 Markdown 和结构化数据 → 渲染 Markdown + 资源量表格 + 价格图表 + 引用源

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色背景（#0a0a0f）+ 紫色主色（#8b5cf6）+ 蓝色辅助光效（#3b82f6）
- **卡片风格**：毛玻璃效果（backdrop-blur），半透明边框（border-white/10）
- **字体**：Display 字体用 Space Grotesk，正文用 Noto Sans SC
- **布局**：桌面端三栏（左侧导航 + 中间主内容 + 右侧日志/引用），移动端上下结构
- **动效**：节点执行流带呼吸光效，页面加载渐入动画，hover 过渡
- **状态色**：success 绿色、warning 黄色、failed 红色、pending 灰色、running 紫色呼吸

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页 | 输入区 | 居中毛玻璃卡片，紫色渐变输入框，圆角生成按钮 |
| 首页 | 示例区 | 3 个可点击卡片，hover 紫色边框 |
| 首页 | 最近报告 | 紧凑列表，左侧状态圆点，右侧时间 |
| 运行页 | 节点进度 | 左侧纵向列表，每个节点含状态图标+名称+消息 |
| 运行页 | 日志面板 | 右侧终端风格，等宽字体，按级别着色 |
| 报告页 | 报告头部 | 标题+时间+操作按钮（复制/下载/重新生成） |
| 报告页 | 价格图表 | ECharts 折线图，紫色主线，虚线均线 |
| 报告页 | 引用源 | 分类卡片，每条含图标+标题+外链箭头 |
| 历史页 | 报告表格 | 深色表格，行 hover 高亮，状态 Badge |

### 4.3 响应式设计

- **桌面端（≥1024px）**：三栏布局，左侧导航 240px + 中间内容自适应 + 右侧面板 320px
- **平板端（768-1023px）**：两栏布局，隐藏左侧导航改为顶部，右侧面板移至下方
- **移动端（<768px）**：单栏上下结构，所有面板垂直排列
